
Projeto Django de exemplo 'Economiza+' com dashboard de usuário, perfil e integração
exemplo com APIs públicas (FakeStoreAPI e OpenFoodFacts). 

Como rodar:
- python -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt
- python manage.py migrate
- python manage.py createsuperuser
- python manage.py runserver
